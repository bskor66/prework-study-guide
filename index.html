HTML

  • The head stores info about the page for the web browser
  • The body stores info for the browser to display to the user
  • Heading tags are important for SEO
  • Only use h1 once per page and cascade down to h2 and so on
  • Alt text is important for accessibility and SEO

CSS

  • You can use inline css, internal sheet, or external sheet
  • External sheet is preferred because it adds a point of access
  • You must link your style sheet in the head of your HTML
  • Padding is the space inside the element
  • Margin is the space outside the element
  • HTML elements work as boxes for css to style
  • Colors can be semantic, hexes, or rgb

Git

  • Always commit new features to a new branch
  • Pull the main branch before pushing to avoid conflicts
  • Open issues for new features
  • git status: check what branch you're in
  • git checkout -b branch-name: Create and switch branch
  • git pull origin main: pull main branch
  • git push origin branch-name: push to branch

JavaScript

  • Javascript is a language to add interactivity and function to your website
  • A variable is a named container that stores data
  • Control Flow is the order in which a computer executes code
  • A for loop can be used to iterate through elements
  • Arrays store multiple values
  • Functions allow you to call the code within them
  • You have to link the script in your code
